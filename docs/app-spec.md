# Budget App v1

## Includes

- Supabase Auth login/register/reset/logout
- CSV statement uploads to Supabase Storage
- Transaction parsing, normalization, dedupe, categorization
- Dashboard metrics and report charts
- Transaction recategorization
- Custom category management

## Stack

- Frontend: Next.js (App Router), JavaScript, Tailwind CSS
- Backend: Node.js + Express
- Database/Auth/Storage: Supabase
